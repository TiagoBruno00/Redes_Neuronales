## Código y Estructura de Notebooks

Esta rama contiene los *notebooks* principales con la implementación y los experimentos. A continuación, se presenta un breve resumen de cada *notebook* incluido en esta rama:

| Notebook | Objetivo Principal | Enfoque |
| :--- | :--- | :--- |
| **`1_1_Clasificador_Consigna.ipynb`** | Guía y *template* para el entrenamiento de un **Clasificador**. | Carga de datos, **preprocesado**, definición de red, entrenamiento y evaluación (métricas y visualizaciones). |
| **`1_Autoencoder_consigna.ipynb`** | Consigna y requerimientos para construir y entrenar un **Autoencoder**. | Arquitectura propuesta, objetivo de **reconstrucción**, procedimiento de entrenamiento y criterios de evaluación. |
| **`1_Autoencoder_lineal_512.ipynb`** | Implementación de un **Autoencoder Lineal**. | Arquitectura con capas densas y **dimensión latente 512**. Entrenamiento, análisis de la **función de pérdida** y ejemplos de reconstrucción. |
| **`2_1_Clasificador_Raschka.ipynb`** | Experimentos de **Clasificación** avanzados. | Implementación inspirada en la **metodología de Raschka**. Incluye ajuste de **hiperparámetros** y evaluación detallada con múltiples métricas. |
| **`2_Autoencoder_128_Raschka.ipynb`** | **Autoencoder** con enfoque avanzado. | Arquitectura con **dimensión latente 128**. Sigue el enfoque de **Raschka**, presentando comparaciones con otras configuraciones y visualizaciones de reconstrucción. |
